# AI Development Rules

Sebelum mengimplementasikan fitur:

1. Selalu lakukan analisis terlebih dahulu.
2. Jangan langsung membuat kode.
3. Tunggu review sebelum implementasi.
4. Pecah pekerjaan menjadi task kecil.
5. Jangan mengubah arsitektur tanpa persetujuan.

---

## Project Rules

- Website publik membaca data dari database.
- Hindari hardcode.
- Gunakan singleton SchoolProfile.
- Jangan menambahkan View Composer kecuali diminta.
- Jangan melakukan redesign dashboard tanpa alasan.
- Sidebar dikembangkan berdasarkan modul.
- Fokus menyelesaikan modul saat ini sebelum membuat modul baru.

---

## Review Rules

Setelah implementasi selalu jelaskan:

- File yang diubah.
- Alasan perubahan.
- Dampak perubahan.
- Langkah pengujian.
